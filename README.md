# Spotify Data Analysis

Exploring what makes music popular using a dataset of 52,000 Spotify songs.

## Projects

### Principles of Data Science — NYU
**Files:** `capstoneProject_Harini_Buddaluru.ipynb`, `.py`, `.pdf`

Statistical analysis and introductory modeling on Spotify audio features:
- Distribution analysis of features like tempo, energy, danceability, and loudness
- Correlation between song length and popularity
- Hypothesis testing (Mann-Whitney U) on explicit vs. non-explicit songs and major vs. minor key songs
- Simple and multiple linear regression to predict song popularity
- PCA on audio features to identify key dimensions of variation
- Logistic regression to classify major/minor key using valence and time signature
- Classical vs. non-classical genre classification using duration and PCA components

### Fundamentals of Machine Learning — NYU
**Files:** `FundamentalsOfML_capstone.ipynb`, `.pdf`

ML modeling on the same Spotify dataset to predict song genre from audio features.

## Dataset

`spotify52kData.csv` — 52,000 Spotify tracks with audio features including danceability, energy, loudness, speechiness, acousticness, instrumentalness, liveness, valence, tempo, and more.
